Title: Resources – MVP
Slug: resources-mvp
Intent: Definire la gestione risorse, assegnazioni e disponibilità.

Requisiti chiave
- RQ1: Creare risorse (resource) e associarle a utenti
- RQ2: Assegnare risorse a task e gestire disponibilità
- RQ3: Verificare integrazione con API Task e Assignment

Acceptance Criteria
- AC1: Risorse possono essere create e assegnate a un task
- AC2: Disponibilità e carico di lavoro visibili nel sistema
- AC3: Accesso per risorse autenticato (in MVP con login base)
